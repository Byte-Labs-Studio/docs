# BL Sprites WIP

A collection of Skill Check minigames for FiveM made in Svelte.

<ResourceLinks repo="https://github.com/Byte-Labs-Project/bl_ui" />

## Installation

<Steps>

### Download the latest [release](https://github.com/Byte-Labs-Project/bl_sprites/releases) of BL Sprites.

### Extract the contents of the zip file into your `resources` folder.

### Add `ensure bl_sprites` to your `server.cfg` file.

### Configure the `client/modules/config.lua` file to your liking.
</Steps>
