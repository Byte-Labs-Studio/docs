# Bridge Interfaces

## Overview

Includes a bridge for the following:
    - Progressbar
    - Notification
    - Context Menu
    - Radial Menu


## Notification

```lua
    Framework.notify({
            description = 'Notification description',
            type = 'success'
        })
```

- description?: `string`
- type?: `'inform'` or `'error'` or `'success'`or `'warning'`
- duration?: `number`

Extra Options:
<Tabs items={["OX"]}>
  <Tab>
    - id?: `string`
        - When set the current notification will be unique and only shown once on screen when spammed.
    - title?: `string`
    - description?: `string`
        - Must provide if there is no title
        - Markdown support
    - Must provide if there is no description
    - position?: `'top'` or `'top-right'` or `'top-left'` or `'bottom'` or `'bottom-right'` or `'bottom-left'` or `'center-right'` or `'center-left'`
    - Default: `'top-right'`
    - style?: `table` (`object`)
    - React CSS styling format
    - icon?: `string`
    - Font Awesome 6 icon name
    - iconColor?: `string`
  </Tab>
</Tabs>

## Context Menu

```lua
    local Context = Framework.context
    @type data: {
            title = string,
            description = string,
            icon = Icon(string, ),
            disabled = boolean,
            event = string,
            args = {xirvin = 'dick'}
    },
    Context.openContext(data)
    Context.closeContext()
```
- title?: `string`
- description: `string`
- icon: `string`
- disabled: `boolean`
- event: `string`
- args: `any`
