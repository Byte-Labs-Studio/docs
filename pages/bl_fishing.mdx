# BL FISHING 

## [**Preview**](https://www.youtube.com/watch?v=bRI-jsMl8_o)
## Installation

<Steps>
### Download the latest from [Keymaster](https://keymaster.fivem.net).

### Download dependencies 
#### [**ox_lib**](https://github.com/overextended/ox_lib/releases/download/v3.27.0/ox_lib.zip)
#### [**bl_bridge**](https://github.com/Byte-Labs-Studio/bl_bridge)
#### [**bl_sprites**](https://github.com/Byte-Labs-Studio/bl_sprites)
#### [**bl_dialog**](https://github.com/Byte-Labs-Studio/bl_dialog/releases/download/v1.1.0/bl_dialog.zip)

### Extract the contents of the zip file into your `resources` folder.

### Add `ensure bl_fishing` and dependencies to your `server.cfg` file.

### Add items images you can find in `bl_fishing/images`
### Add needed items you can find below 
</Steps>
