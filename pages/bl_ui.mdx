# BL UI WIP

A collection of Skill Check minigames for FiveM made in Svelte.

<ResourceLinks repo="https://github.com/Byte-Labs-Project/bl_ui" />

## Installation

<Steps>

### Download the latest [release](https://github.com/Byte-Labs-Project/bl_ui/releases) of BL UI.

### Extract the contents of the zip file into your `resources` folder.

### Add `ensure bl_ui` to your `server.cfg` file.

### Configure the `config.lua` file to your liking.
</Steps>
