# QB Inventory
Below are the steps to add item to your QB inventory.

## Add Item
```lua
["clothes"] = {
    ["name"] = "clothes",
    ["label"] = "Clothes",
    ["weight"] = 1000,
    ["type"] = "item",
    ["image"] = "clothes.png",
    ["useable"] = true,
    ["shouldClose"] = true,
    ["description"] = "A mix of clothes to wear"
}
```
